# Changelog

All notable changes to **meowMieru** (mierukop) are documented here.

## [1.3.1] — 2026-06-25
- **Node-aware**: на mesh-нодах (`/etc/meownet-role` = node) туннель не поднимается —
  интернет ноды идёт через main-роутер по batman-mesh. На обычных роутерах без роли —
  поведение без изменений.
- **Интеграция в прошивку**: модуль (бинарники mieru/hev + CLI + LuCI) встроен в
  overlay сборки вместо podkop/sing-box; first-boot uci-default настраивает сервер из
  provisioned mieru-конфига.

## [1.3.0] — 2026-06-25
- **Многостраничный интерфейс** — вместо одной длинной страницы пять вкладок в
  нативном стиле LuCI: **Обзор** (статус, туннели, трафик, график пинга, качество),
  **Серверы** (выбор активного, таблица серверов, пинг/лучший, подписка),
  **Маршрутизация** (списки, группы, свои правила), **Диагностика** (проверка обхода,
  клиенты), **Настройки** (поведение, автоматизация, сеть, версия/обновление).
- Общий код вынесен в `resources/mierukop/lib.js`; установщик теперь сам ставит все
  файлы LuCI-приложения (раньше требовалась ручная выкладка).

## [1.2.1] — 2026-06-25
First tagged GitHub release. No code changes vs 1.2.0 — published so the built-in
update checker (`mierukop update-check` / «Проверить обновление») has a release to
detect and install.

## [1.2.0] — 2026-06-25
- **Клиенты** — панель устройств LAN: активные соединения, трафик и сколько из них
  идёт через туннель (детект через `ct mark` в nftables).
- **Авто-выбор лучшего сервера** — кнопка «Выбрать лучший сейчас» + тумблер
  авто-режима (cron каждые 15 мин выбирает сервер с наименьшим пингом).
- **Авто-обновление подписки** — URL подписки запоминается при импорте; ежесуточный
  refetch подтягивает новые серверы и сменившиеся IP.
- **Проверка обхода** — по каждому туннелю показывает внешний IP, страну и реально ли
  отвечают YouTube / Telegram / Discord.
- **Карточки туннелей** — статус default + каждой группы с живым внешним IP.
- **График задержки** — спарклайн истории пинга активного сервера.

## [1.1.0] — 2026-06-25
- **Самообновление модуля** — отображение версии, `update-check` (через туннель,
  т.к. raw.githubusercontent заблокирован в РФ), `self-update` одной кнопкой и
  авто-обновление по cron (раз в неделю, по тумблеру).
- Цветовая индикация пинга (зелёный/жёлтый/красный) в таблице серверов и в панели
  качества; бейдж состояния подключения в шапке.

## [1.0.0]
- Первый релиз: селективная маршрутизация через mieru SOCKS5 (podkop-style),
  per-group туннели, подписки в формате Clash, watchdog, kill-switch, LuCI-приложение.
